# Usage

Basic Usage - Move to AR Ads Activity

### Android - Kotlin

1. Import nuMetagenSDK

   ```js
    import id.numetagen.sdk.NuMetagenAdsActivity
   ```

2. Create Intent Activity and Start it.

   ```js
   val i = Intent(this, NuMetagenAdsActivity::class.java)
   startActivity(i)
   ```

### Flutter

1. Import nuMetagenSDK

   ```js
   import "package:numetagen_sdk_flutter/numetagen_sdk_flutter.dart";
   ```

2. And use it on button to open Ads.

   ```js
   ElevatedButton(
      /// This is the button that will call the native platform
      onPressed: () async => await NumetagenSdkFlutter().startAdsActivity(),
      child: const Text('Start Ads Activity'),
   ),
   ```

That's it. Now you can now move to AR Ads Activity.
